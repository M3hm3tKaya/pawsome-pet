# Design System

## Colors
| Name        | Hex       | Usage                    |
|-------------|-----------|--------------------------|
| Warm White  | #FFF9F0   | Background               |
| Orange      | #FF8C42   | Primary / Konaklama      |
| Teal        | #2ABFBF   | Secondary / Gunluk Bakim |
| Pink        | #FF6B8A   | Accent / Spa             |
| Blue        | #5B8DEF   | Accent / Egitim          |
| Dark Brown  | #3D2E24   | Text / Footer BG         |

## Typography
- **Headings**: Quicksand (rounded, friendly)
- **Body**: Nunito (soft, readable)

## Border Radius
- Cards: 16px (rounded-2xl)
- Buttons: 9999px (pill shape)
- Large containers: 24px (rounded-[24px])

## Animations
- Bounce easing: cubic-bezier(0.34, 1.56, 0.64, 1) with 0.5s duration
- Float animation: 6s ease-in-out infinite
- Card hover: translateY(-8px) with bounce easing
- Scroll-triggered: fade up with 40px offset

## Decorative Elements
- PawPrints: Floating SVG paw prints at 0.1-0.15 opacity
- WaveDivider: SVG wave for section transitions
- Color-coded service cards
